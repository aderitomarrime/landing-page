# <p align="center">Landing Page</p>

<p align="center"> <em>A simple landing page with HTML and CSS</em></p>

![Landing page's hero section](https://imgur.com/8QJqXbc.png)

## Overview

This project is a landing page made up of six main sections. It includes a navigation bar, a hero, more information, a quote, a call-to-action and a footer.

## Live Demo

Visit the website: [Live Demo](https://aderitomarrime.github.io/landing-page/)


## Features

- Minimal interface
- One-page layout
- Six sections
    - Navigation bar
    - Hero
    - More information
    - Quote
    - Call-to-action
    - Footer


## Technologies Used

- HTML5  
- CSS3  


## Installation

To run this project locally, follow these steps:

1. Clone the repository:

```sh
git clone https://github.com/aderitomarrime/landing-page.git
```

2. Navigate to the project directory:

```sh
cd landing-page
```
3. Open index.html in your browser


## How to Use

Once the website is open, navigate through the sections to see the different information each one provides.
